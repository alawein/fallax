# Lessons

- Benchmark and scoring copy drifts quickly when it is repeated in multiple
  files. Keep the canonical language in the main repo surfaces and docs lane.
- Dashboard or website layers should not quietly redefine evaluation semantics.
- Reproducibility details are product behavior here, not optional appendix
  material.
- New task packs need regression coverage early or the benchmark stops being a
  stable comparison surface.
